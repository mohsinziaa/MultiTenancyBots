Integration Guide

Multi-Tenant Chatbot Integration

Simple, powerful chatbot integration for your website. Each bot has unique knowledge, branding, and behavior.

📋 Quick Start

Add this single line to your HTML with your specific bot ID:

<script src="https://multi-tenancy-bots.vercel.app/chatbot-widget.js?bot-id=YOUR_BOT_ID" data-auto-init></script>

Replace YOUR_BOT_ID with your actual bot ID from the dashboard.

⚠️ Important:
  • You MUST provide a valid bot ID in the script URL
  • Without a bot ID, the widget will not initialize
  • Ensure your bot is active in the dashboard
  • Test with your bot ID before deploying to production

🔍 How It Works

Each bot gets its own configuration, knowledge base, and branding while sharing the same powerful AI infrastructure.

1

Bot ID Detection

The widget reads the bot ID from the script URL parameter

2

Configuration Fetch

Fetches bot-specific settings, system prompt, and branding

3

Custom Interface

Displays bot-specific name, company, and behavior
